คู่มือการใช้งานระบบเทรด AI ICT/SMC (v5.0) - Scalp Edition
================================================================================

1. ภาพรวมระบบ (System Overview)
-------------------------------
บอทเวอร์ชั่น 5.0 ถูกออกแบบมาเพื่อการเทรดแบบ Scalping โดยเฉพาะ โดยใช้กลยุทธ์ 
Smart Money Concepts (SMC) และ Inner Circle Trader (ICT) ผสานกับแนวคิด 
Price Action พื้นฐาน ระบบนี้ได้นำส่วนของ Grid Trading ออกไปทั้งหมดเพื่อลด 
Drawdown และเพิ่มความเสถียร โดยมี Gemini AI เป็นตัวตัดสินใจหลัก

2. กลยุทธ์การเทรด (Trading Strategy)
-------------------------------
ก. 7-Step ICT Consolidation:
   - ตรวจสอบช่วงราคา Sideways (Consolidation)
   - รอการกวาดสภาพคล่อง (Liquidity Sweep) ออกนอกกรอบ
   - เข้าเทรดเมื่อราคากลับเข้ามาในกรอบ (Market Structure Shift ใน M15)

ข. Trending OTE (Optimal Trade Entry):
   - ค้นหาราคาที่ทำแรงขับเคลื่อนชัดเจน (Impulsive Move)
   - เข้าเทรดที่ระดับ Fibonacci 62% - 79% (OTE Zone)
   - ใช้เป้าหมายกำไรที่ SD 2.0 และ 2.5

3. การตั้งค่าที่สำคัญ (config.py)
------------------------------
- AI_MODEL: ปัจจุบันใช้ "gemini-2.5-flash" (เสถียรที่สุดสำหรับปี 2026)
- AI_CONFIDENCE_THRESHOLD: ตั้งไว้ที่ 70% (ความมั่นใจ AI ก่อนเข้าเทรด)
- LOT_DIVISOR: ตั้งไว้ที่ 10,000 (สูตร: Balance / 10,000)
- MIN_LOT: 0.02 (ขนาด Lot ขั้นต่ำที่บอทจะออก)
- MULTI_TP_RR_LEVELS: [1, 1.5, 2.5] (เป้าหมายกำไรตามระยะ Risk:Reward)

4. ขั้นตอนการติดตั้งและเริ่มใช้งาน
----------------------
1. ตรวจสอบไฟล์ .env: ใส่ GEMINI_API_KEY ของคุณ
2. ติดตั้ง Library: 
   $ pip install -r requirements.txt
3. ตั้งค่า MT5:
   - ตรวจสอบว่าเปิด "Allow Algo Trading"
   - ตรวจสอบว่ามีสัญลักษณ์ "XAUUSDc" หรือคู่เงินที่ต้องการใน Market Watch
4. เริ่มรันบอท:
   $ python trader.py

5. ระบบจัดการความเสี่ยง (Risk Management)
-----------------
- Virtual SL: ระบบเก็บจุดตัดขาดทุนไว้ในเครื่อง ไม่ส่งตัวเลขจริงไปที่โบรกเกอร์ (ป้องกันการโดนล่า SL)
- Global Recovery Mode: หากพอร์ตติดลบเกิน -10% บอทจะเข้าโหมดกู้พอร์ตอัตโนมัติ
- Post-Trade Cooldown: พักการเทรด 5 นาทีหลังจบคู่ เพื่อรอสัญญาณใหม่ที่ชัดเจน
- Session Filter: กรองเวลาการเทรดทองในช่วงที่ตลาดมีความผันผวนสูง (เช่น ช่วงข่าวแรง)

6. การตรวจสอบผลงาน
-----------------
- หน้าจอ Terminal: จะแสดงสถานะ RSI, โซน SMC (Premium/Discount), และความเห็นจาก AI
- Telegram: บอทจะส่งรายงานกำไรทุก 15 นาที และสรุปรายวันทุก 4 ชม.
