================================================================================
📗 คู่มือการติดตั้งและใช้งาน Forex Trading Bot (SMC AI + Grid System)
================================================================================

บอทเทรดตัวนี้ถูกพัฒนาด้วย Python เชื่อมต่อกับ MetaTrader 5 (MT5) โดยมีระบบหลักคือ
Scalp (SMC), Grid Trading, AI Market Analysis และรายงานผลอัตโนมัติ

--------------------------------------------------------------------------------
1. สิ่งที่ต้องมีก่อนการติดตั้ง (Prerequisites)
--------------------------------------------------------------------------------
- Windows OS (แนะนำให้รันบน PC หรือ VPS ที่เปิดตลอด 24/5)
- โปรแกรม MetaTrader 5 (MT5) ติดตั้งและ Login เรียบร้อยแล้ว
- Python 3.10 ขึ้นไป
- โปรแกรม Ollama (สำหรับฟีเจอร์ AI วิเคราะห์ตลาด - ถ้าไม่ใช้สามารถปิดได้)

--------------------------------------------------------------------------------
2. ขั้นตอนการติดตั้ง (Installation)
--------------------------------------------------------------------------------
1. เปิด Terminal (CMD หรือ PowerShell) ในโฟลเดอร์ของโปรเจกต์
2. ติดตั้ง Library ที่จำเป็นผ่าน pip:
   > pip install -r requirements.txt
3. เตรียมโมเดล: ตรวจสอบว่ามีไฟล์ catboost_model.pkl และ rf_model.pkl ในโฟลเดอร์

--------------------------------------------------------------------------------
3. การตั้งค่าระบบ (Configuration)
--------------------------------------------------------------------------------
ตรวจสอบและแก้ไขไฟล์หลักๆ ดังนี้:

- [.env]: ใส่ข้อมูลบัญชี MT5 (Login, Password, Server) 
          และใส่ TOKEN ของ LINE หรือ Telegram เพื่อรับแจ้งเตือน
          *แนะนำให้ใช้ Telegram เพราะส่งได้ไม่จำกัดและฟรีตลอดชีพ*

- [config.py]: แก้ไขตัวแปร ACTIVE_SYMBOL เพื่อเลือกคู่เงินที่จะเทรด
              เช่น "BTCUSDc" หรือ "XAUUSDm"

--------------------------------------------------------------------------------
4. การตั้งค่าการแจ้งเตือน (Notifications)
--------------------------------------------------------------------------------
- LINE: ใส่ Channel Access Token และ User ID (หรือ Group ID)
- Telegram: 
  1. สร้างบอทกับ @BotFather เพื่อเอา Token
  2. หา Chat ID ของคุณจาก @userinfobot
  3. นำมาใส่ในไฟล์ .env

--------------------------------------------------------------------------------
5. วิธีการเริ่มใช้งาน (How to use)
--------------------------------------------------------------------------------
1. เปิดโปรแกรม MetaTrader 5 ทิ้งไว้ (และต้องกดปุ่ม "Algo Trading" ให้เป็นสีเขียว)
2. เปิดโปรแกรม Ollama หากต้องการใช้ AI
3. รันบอทด้วยคำสั่ง:
   > python trader.py

--------------------------------------------------------------------------------
6. ฟีเจอร์ที่สำคัญ (Key Features)
--------------------------------------------------------------------------------
- [AI Market Analysis]: ส่งรายงานวิเคราะห์ตลาดทุกๆ 4 ชม.
- [Performance Summary]: สรุปกำไร/ขาดทุน และ Win Rate ทุกๆ 12 ชม. อัตโนมัติ
- [Reliable Notification]: แจ้งเตือนปิดออเดอร์ (TP/SL) แม่นยำ 100% 
                           (รวมถึงออเดอร์ที่ปิดด้วยมือด้วย)
- [Hybrid Notify]: ส่งแจ้งเตือนเข้าทั้ง LINE และ Telegram พร้อมๆ กัน

--------------------------------------------------------------------------------
7. การแก้ไขปัญหาเบื้องต้น (Troubleshooting)
--------------------------------------------------------------------------------
- [AI Error / Timeout]: แสดงว่า Ollama ในเครื่องทำงานช้า หรือยังไม่ได้เปิดโปรแกรม
- [MT5 Init Failed]: ตรวจสอบว่าเปิดโปรแกรม MT5 และ Login บัญชีไว้แล้วหรือไม่
- [Line 429 Error]: โควตาส่งข้อความ LINE ฟรี (200 ข้อความ) เต็มแล้ว แนะนำให้สลับไปใช้ Telegram
