# 🤖 Dicky2K Pro v1.2.7 - Professional Trading Bot

บอทเทรดอัตโนมัติที่ผสมผสานพลังของ **AI (Llama 3.2)**, **Grid Recovery**, และ **Basket Management** เพื่อการเทรดที่มีประสิทธิภาพสูงสุดบน MetaTrader 5

---

## 🛠 1. การติดตั้ง (Installation)

### ขั้นตอนที่ 1: ติดตั้ง Python
ตรวจสอบว่าคุณมี Python 3.10 ขึ้นไปติดตั้งอยู่ในเครื่อง

### ขั้นตอนที่ 2: ติดตั้ง Library ที่จำเป็น
เปิด Terminal/PowerShell ในโฟลเดอร์โปรเจกต์แล้วรันคำสั่ง:
```powershell
pip install MetaTrader5 PyQt6 requests pytz pandas
```

### ขั้นตอนที่ 3: ติดตั้ง Ollama (สำหรับระบบ AI)
1. ดาวน์โหลดและติดตั้ง [Ollama](https://ollama.com/)
2. รันคำสั่งเพื่อดึงโมเดล: `ollama run llama3.2`

---

## 🚀 2. ฟีเจอร์หลัก (Key Features)

### 🧺 2.1 ระบบ Basket Management (รวบยอด)
- **Basket TP ($):** ปิดออเดอร์ทั้งหมดทันทีเมื่อกำไรรวมถึงเป้าดอลลาร์ที่ตั้งไว้
- **Basket SL ($):** ระบบ Cut Loss ยกแผงเมื่อขาดทุนรวมถึงขีดจำกัด เพื่อรักษาพอร์ต
- **Hidden Stops:** บอทจะไม่ส่งเส้น SL/TP ไปที่โบรกเกอร์ (ตั้งค่าเป็น 0) เพื่อป้องกันการถูกล่า Stop Loss

### 📉 2.2 ระบบ Recovery Grid (ไม้แก้)
- **Grid Step:** ออกไม้แก้เมื่อราคาลากไปถึงระยะห่างที่กำหนด (เช่น 200 pips)
- **Lot Multiplier:** สามารถตั้งค่าตัวคูณไม้แก้ได้ (เช่น 1.0 คือเท่าเดิม, 2.0 คือ Martingale)
- **Max Recovery:** จำกัดจำนวนไม้แก้สูงสุดเพื่อความปลอดภัย

### 🧠 2.3 ระบบ AI Confirmation
- ใช้ **Ollama (Llama 3.2)** วิเคราะห์แนวโน้มตลาดก่อนเปิดไม้แรก
- ช่วยกรองสัญญาณหลอกและเลือกทิศทางที่มีความน่าจะเป็นสูง

### 🚫 2.4 ระบบ News Sentry (หลบข่าว)
- ดึงข้อมูลข่าวกล่องแดง USD จาก **Forex Factory** อัตโนมัติ
- หยุดออกไม้ใหม่ล่วงหน้า 30 นาที และหลังข่าวออก 30 นาที

## ⚙️ 3. การตั้งค่า (Configuration)

แก้ไขไฟล์ `config.json` หรือปรับผ่านหน้า UI:
- `basket_tp_usd`: เป้ากำไรรวม ($)
- `basket_sl_usd`: ขีดจำกัดขาดทุนรวม ($)
- `recovery_step_pips`: ระยะห่างไม้แก้ (Pips)
- `max_spread_pips`: Spread สูงสุดที่อนุญาตให้เทรด
- `min_order_distance_pips`: ระยะห่างขั้นต่ำระหว่างไม้ (ป้องกันไม้ซ้อน)

## 📱 4. การเชื่อมต่อ Telegram
1. ใส่ `telegram_bot_token` และ `telegram_chat_id` ในหน้า Settings
2. กดปุ่ม **Test Telegram Connection** เพื่อยืนยัน
3. บอทจะแจ้งเตือนเมื่อ: *เปิดไม้ใหม่, ปิดรวบยอดกำไร, หรือเข้าสู่ช่วงข่าว*
