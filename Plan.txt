## 1. 🐛 Bug & Vulnerability Hunt

### Critical Execution Failures

| Issue | Severity | Location |
|-------|----------|----------|
| **MT5 Disconnect Recovery Missing** | 🔴 CRITICAL | `init_mt5()` only runs once |
| **Hedge Race Condition** | 🔴 CRITICAL | `check_and_trigger_hedge()` |
| **Loss Shaving Partial Close Broken** | 🟠 HIGH | `shave_losing_positions()` |
| **Virtual SL Cache Unused** | 🟠 HIGH | `self.virtual_sl_cache` never read |
| **AI Hardcoded Endpoint** | 🟡 MEDIUM | `get_ai_market_analysis()` |

### Detailed Findings

**1. No MT5 Reconnection Logic**
```python
# trader.py:97-107
def init_mt5(self):
    if not mt5.initialize():  # ❌ Only called once at startup
        print("MT5 Initialization failed.")
        return False
```
**Risk:** If MT5 disconnects during live trading (broker maintenance, network blip), the EA continues looping blind—no positions managed, no new entries, but Grid/Hedge logic still thinks it's connected.

**Fix:** Add reconnection check in main loop:
```python
def ensure_mt5_connected(self):
    if not mt5.initialize():
        self.notify("🔴 MT5 Disconnected — Attempting reconnect...")
        time.sleep(5)
        return self.init_mt5()
```

**2. Hedge Race Condition**
```python
# trader.py:544-577
def check_and_trigger_hedge(self, symbol, position, atr_val):
    hs = self.hedge_state.get(symbol, {})
    # ❌ Multiple positions can pass this check simultaneously
    if hs.get('hedge_ticket'):
        # ... checks if hedge exists ...
```
**Risk:** If you have 2 BUY positions and price drops 80 pips, both could trigger separate hedge orders before `hedge_state` updates → 4 positions instead of 2.

**Fix:** Check ALL positions before triggering:
```python
all_pos = mt5.positions_get(symbol=symbol) or []
hedge_count = sum(1 for p in all_pos if "HEDGE" in p.comment or "RECOVERY" in p.comment)
if hedge_count > 0:
    return  # Already have hedge active
```

**3. Loss Shaving Market Risk**
```python
# trader.py:771-789
# Closes full position, then reopens remainder
res = mt5.order_send({... "volume":tp.volume ...})  # Full close
# ... market may have moved 10+ pips ...
self._send_order(symbol, reopen_dir, remaining, sl, tp_new...)  # Reopen
```
**Risk:** Between close and reopen, price can gap against you. On XAUUSD during news, this could be 50-100 pips slippage.

**Fix:** Only shave if spread < threshold and volatility is low:
```python
tick = mt5.symbol_info_tick(symbol)
spread = (tick.ask - tick.bid) / pip
if spread > 30:  # Reject if spread too wide
    return
```
